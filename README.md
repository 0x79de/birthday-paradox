# birthday-paradox

A Rust implementation of the classic Birthday Paradox probability simulation.

## Clone & Build

```bash
git clone https://github.com/0x79de/birthday-paradox.git
cd birthday-paradox
cargo build
```

## Usage

```bash
# Default parameters 
cargo run 

# Custom parameters 
# First number = group size, Second number = number of trials
cargo run -- 27 100000
```
